RAF Alumni API

Ovo je jednostavan REST servis čiji API omogućava pristup bazi podataka diplomiranih studenata RAF-a.
Uputstvo za upotrebu servisa je u nastavku 

            GET
            /                       home ruta, prikazuje ovu stranicu
            /student                izlistava sve studente
            /student/find/:string   izlistava studente koji imaju string u imenu, prezimenu, zvanju, temi rada, mentoru
            /student/:id            daje studenta sa zadatim id-em 
            /smer                   izlistava sve smerove
            /smer/:id               izlistava sve studente na tom smeru

            POST
            /student                u body se očekuje json objekat studenta, radi se insert, ispisuje json novog studenta

            PUT
            /student/:id            u body se očekuje json objekat studenta, radi se update, ispisuje json studenta nakon update

            DELETE
            /student/:id            briše studenta po id-u, ispisuje json obrisanog studenta
